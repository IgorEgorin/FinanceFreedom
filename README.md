•Скачайте, установите и запустите Docker desktop у себя локально

•Скачайте весь проект из моего репозитория

•Перейдите в папку скаченного проекта

•Открыть в папке проекта консоль (Powershell, CMD)

•Выполнить "docker build . -t financefreedom-app"

•Выполнить "docker run -it --rm -p 8080:8080 financefreedom-app"


•Результат: Тест успешно пройдет. В консоль в виде текста будет последнее сообщение "Pop-up исчез"
